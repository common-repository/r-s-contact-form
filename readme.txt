== Description ==

With this Ajax compatible contact from, visitors on the website can connect with you easily. The form has advanced features that will allow you to compose, reply, keep draft and connect in a structured way. With the built in control options and design presets, you can bring variation in the overall structure of the form.

== Installation ==

There are a few options for installing and setting up this plugin.

= Upload Manually =

1. Download and unzip the plugin.
2. Upload the 'rs-contact-form' folder into the '/wp-content/plugins/' directory
3. Go to the Plugins admin page and active the plugin.

= Install Via Admin Area =

1. In the admin area go to Plugins > Add New and search "R S Contact Form"
2. Click install and then activate.

= How to Use =

1. Configure setting from plugin settings (Message -> Settings)
2. Configure SMTP setting
3. Notice Message
4. Use '[show_contact_form]' where to display form
